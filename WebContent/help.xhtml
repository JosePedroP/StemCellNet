StemCellNet

What is StemCellNet ?


StemCellNet is an interactive platform for retrieval and analysis of molecular networks associated with stem cells and their marker genes. It contains a large number of physical and regulatory interactions both from human and mouse, which can be queried by the user through the input of a set of gene or protein identifiers. The web-interface includes various filtering options for prioritization, and allows the upload of own data for display of differential expression. As a unique feature of StemCellNet, the obtained networks can be rapidly screened for genes included in various published stemness signatures. This feature can also be applied to dissect pathogenetic processes, in which stem cell related mechanisms play key roles, such as in degenerative diseases and oncogenesis.

Tutorial for StemCellNet


In the following, we describe the main features of StemCellNet using as input set CITED1, a transcriptional co-activator, and the growth arrest and DNA-damage-inducible proteins GADD45A and GADD45G. The set can be obtained as input by pressing the Example button.


Selecting central nodes


The first step in using StemCellNet is simply to input of a list of gene identifiers (including include gene name, gene symbol, Entrez Gene ID, Uniprot ID and Ensembl ID for human or mouse) in the search box and pressing the Search button. The inputted identifiers are then matched against the list of genes/proteins included in StemCellNet. For the matches, the genes symbol, their description, aliases and various identifiers linked to other databases are given. Furthermore, a set of buttons is displayed indicating the original source of interactions for the found genes (Figure 1).


Figure 1. List of genes found in StemCellNet.


Once a list of genes is obtained, the next step is to select which ones will serve as central nodes. The network will be constructed around these central nodes based on the interactions included in StemCellNet for them. After selecting the central nodes, press Draw Network.


Analyzing the network


As shown in Figure 2, the page, which renders the retrieved interaction network and serves as interface for network-orientated analyses in StemCellNet, is divided into two panels: the left one for the graphical display of the network and the right one listing the available tools. The later one is further subdivided based in functionality, which will be described below.

In the default display, the central nodes are larger and marked in grey whereas the interacting proteins as shown as yellow nodes. Protein-protein interactions are represented as blue edges, while transcriptional regulatory interactions by red ones.


Stemness gene sets


StemCellNet includes currently 20 gene sets, which have previously been linked to stemness (i.e. to the characteristic feature of stem cells: the capacity to both self-renew and differentiate into mature cells). These gene (also referred to as stemness signatures) sets have been derived from three different types of sources: (i) ChIP-chip experiments detecting activated target genes of the core transcription factors Nanog, Pou5f1 (Oct4), and Sox2, or co-activated by all three transcription factors (abbreviated here as NOS targets), (ii) gene expression studies to identify up-regulated genes in stem cells compared to other types of (differentiated) cells and (iii) large-scale functional RNAi screens to detect genes whose knock-down leads to loss of stem cell markers. A table with an overview of the included stemness gene sets with references to the published studies can be seen here.


To map stemness gene sets onto a network, you can select the relevant gene set by activating the check box and press the Map gene sets button. The size of the nodes is then adjusted to be proportional to the number of stemness gene sets, in which the corresponding gene appears. Nodes which represent genes that do not appear in any stemness gene set are displayed as rectangles.


As an alternative to visualization these results can also be downloaded in the format CSV using the Download set table button. Here, a table is outputted with the genes and their occurrence in stemness signatures. It should be noted that the file will only contain data about nodes which are present in at least one set.


Figure 3. Network after mapping the gene sets. Nodes corresponding to genes, such as the key transcription factors POU5F1, NANOG or SOX2, which appear in many stemness gene sets, are noticeably increased in size


In some cases, it might be interest to identify novel candidates genes associated with stemness. To assist researchers in this gene prioritization problem StemCellNet has a tool that evaluates the connectivity within the network and scores potential candidates accordingly. Currently, four variations of the ‘direct neighbour counting’ approach are available: unweighted, weighted scoring, weighted by co-expression scoring and weighted by co-expression and set number scoring. These methods are motivated on the ‘guilt by association’ principle and assume in our context that the more stemness genes that a candidate gene is directly connected to, the more likely it is that the candidate itself is associated with stemness.


The Unweighted connectivity button calls the unweighted scoring version of this function in which the score of a candidate is simply the sum of direct interactions with stemness associated genes. Alternatively by using the Weighted connectivity button each interaction is weighted by the number how many time the interacting genes occurs in a stemness signature. Note that for using either option a group of stemness sets must be mapped first into the network (by pressing the Map gene sets sets button).


Two related methods which utilize co-expression between interacting genes are the (i) Weighted by co-expression and (ii) Weighted by co-expression and set count, which can be executed by respective buttons. Both methods are variants of the unweighted connectivity and weighted connectivity scoring respectively, as the each interaction is additionally weighted by the correlation coefficient for co-expression. Basically, the scores are calculated as the sum of (weighted) interactions. Additionally, each factor representing an interaction between two genes is multiplied by the correlation coefficient for these two genes. The data set, from which the correlation coefficients are derived, can be selected by the user in a drop-down menu. Note that only interactions with positive correlation of expression contribute to the final score: When the correlation coefficient is positive the adjustment is calculated by multiplying the correlation coefficient by 10 and normalizing the results, so that the decimal is removed. In contrast, when the correlation coefficient is zero or negative, the terms representing these interactions are multiplied by 0.


Significance of enrichment


Enrichment of genes from the stemness sets in the network can be calculated by pressing the Calculate enrichment button. It is not necessary to map the stemness sets onto the network before calculating the enrichment. The calculation is based on the Fisher’s exact test. For each stemness gene set, a p-value is displayed for the significance together with the number of genes found within the network.

It is important to note that the calculation does not take into account nodes, which were filtered out of the network (using the functionality described below). Also, the values are not automatically updated, when a filter is applied. In that situation the function has to be manually recalled.


Gene expression


Gene expression data can be uploaded into StemCellNet as tab delimited table with the following format:

1st column: Entrez Gene ID or Gene Symbol.

2nd column: Expression values (such as signal intensities or fold changes).

3rd column: P-value.

4rt column: Sample description.


Once the file is uploaded, the nodes can be colored based on a user-defined cutoff values for either the expression values or the p-value. In the case of expression value, nodes associated with a smaller value are colored green, while nodes with a larger value are colored red. Nodes with no associated values keep their original coloring. For filtering based on p-values, only nodes with an p-value smaller than the chosen cutoff are highlighted. Here, nodes passing the threshold are colored red if they have positive expression, whereas they appear green for negative expression. This filtering option can be used in the common cases where the expression values in the file provide differential expression given as logged fold changes. In either situation yellow nodes can optionally be filtered out of the network.


It should be noted that files can contain data from several different samples assuming that each has a unique sample id. This enables the simultaneous uploading of expression data for multiple samples and thus facilitates the analysis of more complex experiments such as time series.


A exemplary file compatible with the StemCellNet format can be downloaded here. This file provides an excerpt from a time series experiment, in which mouse embryonic stem cells have undergone differentiation through the hanging droplet method. Note that three time points are included: day 0, day 3 and day 10. Once the file is uploaded, the time point of interest can be used for filtering by choosing it in the drop down menu.

Filters


StemCellNet creates networks with both human and mouse interactions. However if only interactions specific for one of the species are of interest, it is possible to remove interaction from the other organism though either the Remove non human or Remove non mouse buttons.


This option removes foremost edges. In case that a node is connected only with edges which have been removed, it will also be removed.


Additionally, StemCellNet provides filtering options for removing either physical or regulatory interactions.


Also it is possible to filter the interactions based in the number of associated Pubmed ids. In this case a threshold must be set before using the filter. The filtering removes all interactions which have been associated with less publications than the chosen threshold. This functionality can help to derive high-confidence networks.


Co-expression filters are based in the expression data stored in the associated database (Table 2). This filters can be used to either remove all the interactions which have a correlation coefficient value less than a chosen threshold or to map the values into the network, in the last case the width of the edges will become proportional to the associated correlation coefficient. Regardless of the option selected this types of filters always remove the edges with no associated correlation coefficient.


Finally this toolbar also contains a functionality which can alter the size of the edges based in the number of associated Pubmed ids. While not a filter this operation can be used to give an idea of the confidence of the network.


Download network


StemCellNet provides options to download the displayed either as graphics in pdf or png format or as a list of interactions in a tab delimited file.

Note that the tab delimited file will only contain the interactions for the currently displayed network.
