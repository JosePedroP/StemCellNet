StemCellNet
Welcome to StemCellNet, a web-server that allows you to interactively query, visualize and interrogate molecular networks in the context of stem cell biology. StemCellNet includes a comprehensive set of physical and regulatory interactions for both human and mouse. Any retrieved network can be readily examined for the presence of genes linked to stemness. An overview of the included data can be found here. To start your analysis, please insert your genes of interest in the input field below. Currently, a maximum number of 500 genes are allowed. Larger queries will be automatically truncated. Pressing the "Search" button will produce the list of genes, for which interactions were found in the current version of StemCellNet. All of these interactions can be downloaded. The displayed list also allows to select up to 20 genes, which will serve as central nodes in the network to be visualized and interactively analysed. If this is your first time using StemCellNet, you might want to check the help page and use the “Example” button to obtain an exemplary input for StemCellNet.





   



If you have any problems using StemCellNet or wish to suggest new data sets to be included please do not hesitate in contact us.

StemCellNet is described in the Nuclei Acids WebServer Issue 2014.
